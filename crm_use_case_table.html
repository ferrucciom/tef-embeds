1 — use cases vs. tool scenarios (8 seats, 7k contacts, VAT incl., annual billing)
Use case HubSpot Attio Pro + Mailchimp Custom on Airtable
Network management Contacts, companies, relations, Outlook sync, call recordings, reminders
  • Built-in contacts, companies, etc. — equivalent to Attio
  • As complexity increases, custom objects can be purchased separately
  • Built-in contacts, companies, etc. — Attio equivalent to HubSpot
  • As complexity increases, custom objects are included in the costs
  • Airtable to help managing/filtering contacts and people, with some frontend for viz and some custom layer to capture and manage Outlook emails
Program execution Participants, cohorts, session attendance, milestones, resources received, interactions
  • Basic relations are doable; complexity and costs increase as data model grows
  • Programs, cohorts, editions, participants, and interactions can be mapped as custom objects — session attendance, milestones, and resources received can be modelled natively
  • We can use Airtable and complement it with additional views — similar to what we are doing with Ignition
Email campaigns Newsletters, event invitations, program communications, community building
  • Native. Segmentation, multi-step automation, and analytics in one platform
  • Attio does not support mass communications — Mailchimp Standard required as dedicated tool
  • 2 tools = 2 logins, 2 contact lists, and a sync dependency to be maintained with 3rd party integrations
  • Two cases: either use Mailchimp anyway, or use Microsoft APIs to build a small email send service (tracking which emails are delivered and opened will require additional work)
Funnel management Awareness to conversion, sales cycles — fundraising, top-of-funnel startup programs
  • Pipeline and deal stages native. Multiple pipelines with custom stages; marketing engagement data feeds contact and deal records automatically
  • Pipelines are native. Multiple pipelines with custom stages; interactions data don't flow back to records natively. Funnel incomplete without custom integrations to connect Mailchimp and Attio
  • Airtable + some middle layer and frontend to integrate same features of current solution and/or Sara's work as frontend
Contact engagement intelligence All interaction signals — program participation, emails, events, web, social — unified on each contact record
  • Native. Interactions (program participation, email opens, event attendance, web and social media signals) aggregated on contact timelines
  • No unified engagement layer across Attio and Mailchimp — signals from campaigns, events, and programs must be assembled via custom integrations; incomplete out of the box
  • Reminders and automations can be set up in Airtable and/or a custom logic layer with some frontend to filter/personalise
